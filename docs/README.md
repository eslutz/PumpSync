# PumpSync iOS Documentation

The shared project wiki is the single source of truth for cross-repository documentation:

- [PumpSync wiki](https://github.com/eslutz/PumpSync/wiki)

This iOS repository keeps app-owned source documents:

- `app-store/` - App Store evidence, accessibility answers, and screenshot support.
- `legal/` - privacy policy, terms, data deletion request, and App Store privacy notes.

Backend API, self-hosting, Azure Container Apps, Docker, and data deletion tooling documentation belongs in [`eslutz/PumpSync-Backend`](https://github.com/eslutz/PumpSync-Backend) and is summarized from the wiki.
